ViewContext.scripts
===================

Возвращает ассоциативный список `скриптов
представления <../Script/>`__.

Syntax
------

.. code:: js

    ViewContext.scripts

Examples
--------

.. code:: js

    var function1 = ViewContext.scripts.myFunction;
    var function2 = ViewContext.scripts['myFunction']; // function2 === function1
