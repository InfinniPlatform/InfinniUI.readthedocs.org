View.getDialogResult()
======================

Возвращает `результат работы представления <../DialogResult/>`__.

Syntax
------

.. code:: js

    view.getDialogResult()

Parameters
~~~~~~~~~~

Нет

Returns

`Результат работы представления <../DialogResult/>`__.

Examples

    var dialogResult = view.getDialogResult();
