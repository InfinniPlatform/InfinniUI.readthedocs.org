DialogResult
============

Результат работы диалогового окна.

Syntax
------

.. code:: js

    DialogResult = {
      none: 0,
      accepted: 1,
      canceled: 2
    }

Members
~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Description
   * - none
     - Не определен
   * - accepted
     - Пользователь подтвердил диалог
   * - canceled
     - Пользователь отменил диалог
