Element.getBackground()
=======================

Возвращает `цвет </docs/API/Core/Style/ColorStyle/>`__ фона элемента.

Syntax
------

.. code:: js

    Element.getBackground()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`Цвет </docs/API/Core/Style/ColorStyle/>`__ фона элемента.

Examples
--------

.. code:: js

    var background = Element.getBackground();
