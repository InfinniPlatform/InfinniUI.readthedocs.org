Element.getName()
=================

Возвращает наименование элемента.

Syntax
------

.. code:: js

    Element.getName()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Наименование элемента.

Examples
--------

.. code:: js

    var elementName = Element.getName();
