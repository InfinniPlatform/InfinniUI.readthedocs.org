Element.getTextStyle()
======================

Возвращает `стиль текста </Core/Style/TextStyle/>`__ элемента.

Syntax
------

.. code:: js

    Element.getBackground()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`Стиль текста </Core/Style/TextStyle/>`__ элемента.

Examples
--------

.. code:: js

    var textStyle = Element.getTextStyle();
