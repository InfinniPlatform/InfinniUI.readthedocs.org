Element.getValidationState()
============================

Возвращает валидационный статус элемента.

Syntax
------

.. code:: js

    Element.getValidationState()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Валидационный статус элемента.

Examples
--------

.. code:: js

    var validationState = Element.getValidationState();

--------
