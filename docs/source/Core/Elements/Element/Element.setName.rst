Element.setName()
=================

Устанавливает наименование элемента.

Syntax
------

.. code:: js

    Element.setName(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``String``
     - Наименование элемента


Returns
~~~~~~~

Метод не возвращает значений.

Examples
--------

.. code:: js

    Element.setName('button1');

--------
