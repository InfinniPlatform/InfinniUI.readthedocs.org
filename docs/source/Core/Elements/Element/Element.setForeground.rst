Element.setForeground()
=======================

Устанавливает `цвет </docs/API/Core/Style/ColorStyle/>`__ текста
элемента.

Syntax
------

.. code:: js

    Element.setForeground(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``String``
     - `Цвет </docs/API/Core/Style/ColorStyle/>`__ текста элемента.


Returns
~~~~~~~

Метод не возвращает значений.

Examples
--------

.. code:: js

    Element.setForeground(ColorStyle.primary1);
