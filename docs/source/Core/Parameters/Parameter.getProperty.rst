Parameter.getProperty
=====================

Возвращает значение атрибута параметра.

Syntax
------

.. code:: js

    Parameter.getProperty(name);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``name``
     - ``String``
     - наименование параметра.


Returns
-------

Значение атрибута параметра.

Examples
--------

.. code:: js

    var firstName = Parameter.getProperty('first_name');

--------
