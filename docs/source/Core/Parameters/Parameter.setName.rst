Parameter.setName
=================

Устанавливает наименование параметра.

Syntax
------

.. code:: js

    Parameter.setName(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``String``
     - Наименование параметра.


Returns
-------

Нет

Examples
--------

.. code:: js

    Parameter.setName('parameter1');

--------
