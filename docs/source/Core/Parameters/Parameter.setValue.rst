Parameter.setValue
==================

Устанавливает значение параметра.

Syntax
------

.. code:: js

    Parameter.setValue(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - \*
     - Значение параметра.


Returns
-------

Нет

Examples

.. code:: js

    Parameter.setValue(33);
