ServerAction.getParam()
=======================

Возвращает значение параметра.

Syntax
------

.. code:: js

    serverAction.getParam(name)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Description
   * - name
     - Название параметра


Returns
~~~~~~~

Значение указанного параметра.

Examples
--------

.. code:: js

    var documentName = serverAction.getParam('documentName');

--------

-  `setParam <ServerAction.setParam.html>`__
