RouteToAction
=============

Производит переход по указанному имени маршрута. Путь соотвествующий
имени определен в
`InfinniUI.config.Routes <../../InfinniUI/InfinniUI.config.Routes.html>`__.

Extends
-------

`BaseAction <../BaseAction/>`__

Syntax
------

.. code:: js

    new RouteToAction()

Parameters
~~~~~~~~~~

Нет.

Properties
----------

Нет

Examples
--------

.. code:: js

    var routeToAction = new RouteToAction();
    routeToAction.execute();
