RouteToAction
=============

Производит переход по указанному имени маршрута. Путь соотвествующий
имени определен в
`InfinniUI.config.routes <../../Routing/Routes/>`__.

Extends
-------

`BaseAction <../BaseAction/>`__

Syntax
------

.. code:: js

    new RouteToAction()

Parameters
~~~~~~~~~~

Нет.

Properties
----------

Нет

Examples
--------

.. code:: js

    var routeToAction = new RouteToAction();
    routeToAction.execute();
