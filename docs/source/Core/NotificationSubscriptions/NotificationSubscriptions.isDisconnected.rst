NotificationSubscriptions.isDisconnected
========================================

Возвращает значение, разорванно соединение с сервером signalR или нет.

Syntax
------

.. code:: js

    NotificationSubscriptions.isDisconnected()

Parameters
~~~~~~~~~~

Нет.

Returns
~~~~~~~

true или false.

Examples
--------

.. code:: js

    NotificationSubscriptions.isDisconnected();
