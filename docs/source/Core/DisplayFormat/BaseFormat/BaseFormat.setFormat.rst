BaseFormat.setFormat
====================

Устанавливает строку форматирования значения.

Syntax
------

.. code:: js

    Format.setFormat(value);

Parameters
~~~~~~~~~~

+-------------+--------------+-----------------------------------+
Name        | Type         | Description                       |
+=============+==============+===================================+
``value``   | ``String``   | Строка форматирования значения.   |
+-------------+--------------+-----------------------------------+

Returns

Нет.
