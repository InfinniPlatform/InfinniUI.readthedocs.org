BooleanFormat.setTrueText
=========================

Устанавливает текст для отображения истинного значения.

Syntax
------

.. code:: js

    Format.setTrueText(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``String``
     - Текст для отображения истинного значения.


Returns
-------

Нет.

Examples
--------

.. code:: js

    format.setTrueText('Да');
