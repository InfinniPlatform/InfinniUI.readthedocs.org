BooleanFormat.getTrueText
=========================

Возвращает текст для отображения истинного значения.

Syntax
------

.. code:: js

    Format.getTrueText();

Parameters
~~~~~~~~~~

Нет

Returns
-------

Текст для отображения истинного значения.

Examples
--------

.. code:: js

    format.getTrueText();
