BooleanFormat.getFalseText
==========================

Возвращает текст для отображения ложного значения.

Syntax
------

.. code:: js

    Format.getFalseText();

Parameters
~~~~~~~~~~

Нет

Returns
-------

Текст для отображения ложного значения.

Examples
--------

.. code:: js

    format.getFalseText();
