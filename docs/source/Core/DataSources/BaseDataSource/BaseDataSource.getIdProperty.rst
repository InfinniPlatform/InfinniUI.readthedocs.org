BaseDataSource.getIdProperty()
==============================

Возвращает наименование свойства с идентификатором элемента.

Syntax
------

.. code:: js

    BaseDataSource.getIdProperty()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Наименование свойства с идентификатором элемента.

Examples
--------

.. code:: js

    var idProperty = BaseDataSource.getIdProperty();
