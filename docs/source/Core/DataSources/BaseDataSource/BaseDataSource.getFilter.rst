BaseDataSource.getFilter()
==============================

Возвращает правило фильтрации документов.

Syntax
------

.. code:: js

    BaseDataSource.getFilter()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Правило фильтрации документов (см.
`формат <BaseDataSource.setFilter/#filter-format>`__)

Examples
--------

.. code:: js

    var filter = dataSource.getFilter();
