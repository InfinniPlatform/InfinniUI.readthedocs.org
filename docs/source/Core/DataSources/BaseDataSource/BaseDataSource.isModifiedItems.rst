BaseDataSource.isModifiedItems()
================================

Проверяет, есть ли несохраненные элементы.

Syntax
------

.. code:: js

    BaseDataSource.isModifiedItems()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Возвращает ``true``, если есть несохраненные элементы, иначе возвращает
``false``.

Examples
--------

.. code:: js

    var haveModifiedItems = BaseDataSource.isModifiedItems();
