DocumentDataSource.getDocumentId()
==================================

Возвращает идентификатор документа.

Syntax
------

.. code:: js

    DocumentDataSource.getDocumentId()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Идентификатор документа.

Examples
--------

.. code:: js

    var documentId = DocumentDataSource.getDocumentId();
