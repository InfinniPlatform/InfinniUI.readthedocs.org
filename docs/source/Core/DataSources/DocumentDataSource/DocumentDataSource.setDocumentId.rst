DocumentDataSource.setDocumentId()
==================================

Устанавливает идентификатор документа.

Syntax
------

.. code:: js

    DocumentDataSource.setDocumentId(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Description
   * - value
     - Идентификатор документа


Returns
~~~~~~~

Метод ничего не возвращает

Examples
--------

.. code:: js

    DocumentDataSource.setDocumentId('MyDocument');

--------
