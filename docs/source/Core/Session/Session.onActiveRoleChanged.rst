Session.onActiveRoleChanged
===========================

Устанавливает обработчик события изменения активной роли текущего
пользователя.

Syntax
------

.. code:: js

    Session.onActiveRoleChanged(callback);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``callback``
     - ``Function``
     - Обработчик события изменения активной роли текущего пользователя.


Returns
-------

Нет
