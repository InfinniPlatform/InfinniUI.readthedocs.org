ScriptBuilder
=============

`Билдер <../Builder/>`__ скриптов. По переданным метаданным скрипта
создает `скрипт <..>`__ (js-функция, принимающая два параметра: контекст
и аргументы).

Parameters
~~~~~~~~~~

При создании экземпляров ``ScriptBuilder`` параметры не требуются.
