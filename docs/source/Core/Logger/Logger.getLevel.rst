Logger.getLevel
===============

Возвращает минимальный `уровень регистрируемых
сообщений <LogLevel/>`__.

Syntax
------

.. code:: js

    Logger.getLevel()

Returns
~~~~~~~

Минимальный `уровень регистрируемых сообщений <LogLevel/>`__.

Examples
--------

.. code:: js

    var level = Logger.getLevel();

--------

-  `LogLevel <LogLevel/>`__
