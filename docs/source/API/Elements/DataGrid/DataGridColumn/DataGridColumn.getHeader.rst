DataGridColumn.getHeader
========================

Возвращает заголовок колонки.

Syntax
------

.. code:: js

    DataGridColumn.getHeader();

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Заголовок колонки.

Examples
--------

.. code:: js

    var header = DataGridColumn.getHeader();
