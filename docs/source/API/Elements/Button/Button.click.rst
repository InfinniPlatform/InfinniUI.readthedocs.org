Button.click
============

Осуществляет программный клик по кнопке.

Syntax
------

.. code:: js

    button.click()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Метод ничего не возвращает.

Examples
--------

.. code:: js

    button.click();
