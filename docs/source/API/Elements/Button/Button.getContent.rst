Button.getContent
=================

Возвращает содержимое кнопки, установленное методом
`setContent <../Button.setContent.html>`__.

Syntax
------

.. code:: js

    button.getContent()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Содержимое кнопки.

Examples
--------

.. code:: js

    var content = button.getContent();

See Also
--------

-  `setContent <../Button.setContent.html>`__
