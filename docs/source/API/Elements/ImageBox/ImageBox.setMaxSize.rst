ImageBox.setMaxSize
===================

Устанавливает максимальный размер данных в байтах.

Syntax
------

.. code:: js

    ImageBox.setMaxSize(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``Number``
     - Максимальный размер данных в байтах


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    imageBox.setMaxSize(1000000);
