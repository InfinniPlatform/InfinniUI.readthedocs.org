DateTimePicker.setTimeZone
==========================

Возвращает смещение часового пояса. Смещением является разность в
минутах между временем UTC и местным временем.

Syntax
------

.. code:: js

    DateTimePicker.пetTimeZone()

Parameters
~~~~~~~~~~

Нет
