ToggleButton.setTextOn
======================

Устанавливает текст на включенное состояние.

Syntax
------

.. code:: js

    toggleButton.setTextOn(value)

Parameters
~~~~~~~~~~

+-------------+--------------+----------------------------------+
Name        | Type         | Description                      |
+=============+==============+==================================+
``value``   | ``string``   | текст на включенное состояние.   |
+-------------+--------------+----------------------------------+

Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    toggleButton.setTextOn("ON");
