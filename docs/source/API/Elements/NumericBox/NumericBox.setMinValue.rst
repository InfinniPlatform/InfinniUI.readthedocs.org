NumericBox.setMinValue
======================

Устанавливает минимальное значение для ввода.

Syntax
------

.. code:: js

    NumericBox.setMinValue(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``Number``
     - Минимальное значение для ввода.


Examples
--------

.. code:: js

    numericBox.setMinValue(0);
