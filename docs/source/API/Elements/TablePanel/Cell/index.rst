Cell
====

Ячейка таблицы `TablePanel <../>`__ или сетки
`GridPanel <../../GridPanel/>`__.

Description
-----------

Ширину ячейки можно определить с помощью метода
`setColumnSpan <Cell.setColumnSpan.html>`__. По умолчанию ширина равна
``1``.
