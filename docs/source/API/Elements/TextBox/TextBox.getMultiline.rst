TextBox.getMultiline
====================

Возвращает значение, определяющее, разрешен ли многострочный текст.

Syntax
------

.. code:: js

    textbox.getMultiline()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Значение, определяющее, разрешен ли многострочный текст.

Examples
--------

.. code:: js

    var isTextArea = textbox.getMultiline();
