TextBox.setLineCount
====================

Устанавливает максимально видимое количество строк.

Syntax
------

.. code:: js

    textbox.setLineCount(lineCount)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Default
     - Description
   * - lineCount
     - ``Integer``
     - -
     - Максимально видимое количество строк


Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    textbox.setLineCount(10);
