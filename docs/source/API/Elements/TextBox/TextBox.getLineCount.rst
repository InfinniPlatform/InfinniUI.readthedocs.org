TextBox.getLineCount
====================

Возвращает максимально видимое количество строк.

Syntax
------

.. code:: js

    textbox.getLineCount()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Возвращает максимально видимое количество строк.

Examples
--------

.. code:: js

    var lineCount = textbox.getLineCount();
