DataNavigation.getPageCount
===========================

Возвращает количество страниц.

Syntax
------

.. code:: js

    DataNavigation.getPageCount()

Returns
~~~~~~~

Количество страниц.

Examples
--------

.. code:: js

    var pageCount = dataNavigation.getPageCount();
