DataNavigation.getPageNumber
============================

Возвращает номер текущей страницы.

Syntax
------

.. code:: js

    DataNavigation.getPageNumber()

Returns
~~~~~~~

Номер текущей страницы.

Examples
--------

.. code:: js

    var pageNumber = dataNavigation.getPageNumber();
