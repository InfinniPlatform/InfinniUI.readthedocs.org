DocumentDataSource.getPageSize()
================================

Возвращает размер страницы.

Syntax
------

.. code:: js

    DocumentDataSource.getPageSize()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Размер страницы. Положительное число.

Examples
--------

.. code:: js

    var pageSize = DocumentDataSource.getPageSize();
