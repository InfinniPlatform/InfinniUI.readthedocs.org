DocumentDataSource.getFilter()
==============================

Возвращает правило фильтрации документов.

Syntax
------

.. code:: js

    DocumentDataSource.getFilter()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Правило фильтрации документов (см.
`формат <../DocumentDataSource.setFilter/#filter-format>`__)

Examples
--------

.. code:: js

    var filter = DocumentDataSource.getFilter();
