ServerAction.setParam()
=======================

Устанавливает значение параметра.

Syntax
------

.. code:: js

    serverAction.setParam(name, value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Description
   * - name
     - Название параметра
   * - value
     - Значение, которое необходимо установить


Returns
~~~~~~~

Метод ничего не возвращает

Examples
--------

.. code:: js

    serverAction.setParam('documentName', 'Patient');
