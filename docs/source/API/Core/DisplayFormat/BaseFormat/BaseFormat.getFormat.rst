BaseFormat.getFormat
====================

Возвращает строку форматирования значения.

Syntax
------

.. code:: js

    Format.getFormat();

Parameters
~~~~~~~~~~

Нет

Returns
-------

Строка форматирования значения.

Examples
--------

.. code:: js

    format.getFormat();
