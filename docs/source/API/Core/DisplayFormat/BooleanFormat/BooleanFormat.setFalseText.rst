BooleanFormat.setFalseText
==========================

Устанавливает текст для отображения ложного значения.

Syntax
------

.. code:: js

    Format.setFalseText(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``String``
     - Текст для отображения ложного значения.


Returns
-------

Нет.

Examples
--------

.. code:: js

    format.setFalseText('Нет');
