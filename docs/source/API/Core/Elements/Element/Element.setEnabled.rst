Element.setEnabled()
====================

Устанавливает значение, определяющее, возможен ли доступ к элементу или
он заблокирован.

Syntax
------

.. code:: js

    Element.setEnabled(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Логическое значение, определяющее, возможен ли доступ к элементу или он заблокирован.


Returns
~~~~~~~

undefined

Examples
--------

.. code:: js

    Element.setEnabled(true);

--------
