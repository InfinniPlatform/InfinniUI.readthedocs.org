MouseEvent
==========

Событие от мыши.

Syntax
------

.. code:: js

    new MouseEvent()

Properties
----------

``source``
~~~~~~~~~~

Источник события - `визуальный элемент <../>`__.

``button``
~~~~~~~~~~

Возвращает код нажатой кнопки мыши.

``altKey``
~~~~~~~~~~

Возвращает ``true``, если событие произошло, когда была нажата кнопка
``Alt``, иначе - ``false``.

``ctrlKey``
~~~~~~~~~~~

Возвращает ``true``, если событие произошло, когда была нажата кнопка
``Ctrl``, иначе - ``false``.

``shiftKey``
~~~~~~~~~~~~

Возвращает ``true``, если событие произошло, когда была нажата кнопка
``Shift``, иначе - ``false``.

``nativeEventData``
~~~~~~~~~~~~~~~~~~~

Возвращает MouseEvent Object.
