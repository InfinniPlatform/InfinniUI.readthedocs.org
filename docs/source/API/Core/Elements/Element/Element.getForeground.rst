Element.getForeground()
=======================

Возвращает `цвет </docs/API/Core/Style/ColorStyle/>`__ текста элемента.

Syntax
------

.. code:: js

    Element.getForeground()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`Цвет </docs/API/Core/Style/ColorStyle/>`__ текста элемента.

Examples
--------

.. code:: js

    var foreground = Element.getForeground();
