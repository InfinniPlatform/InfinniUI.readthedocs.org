Element.setText()
=================

Устанавливает текст элемента.

Syntax
------

.. code:: js

    Element.setText(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``String``
     - Текст элемента.


Returns
~~~~~~~

Метод не возвращает значений.

Examples
--------

.. code:: js

    Element.setText('Text');

--------
