View.getApplicationView()
=========================

Возвращает корневое представление, т.е. представление, у которого нет
родительского элемента.

Syntax
------

.. code:: js

    view.getApplicationView()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Корневое представление.

Examples
--------

.. code:: js

    var rootView = view.getApplicationView();
