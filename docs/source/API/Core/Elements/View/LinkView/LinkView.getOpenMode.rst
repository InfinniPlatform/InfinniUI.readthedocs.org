LinkView.getOpenMode()
======================

Возвращает `режим отображения представления <../OpenMode/>`__.

Syntax
------

.. code:: js

    linkView.getOpenMode()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Режим отображения представления.

Examples
--------

.. code:: js

    var openMode = linkView.getOpenMode();
