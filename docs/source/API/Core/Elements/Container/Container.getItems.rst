Container.getItems()
====================

Возвращает `коллекцию <../../../Collection/>`__ дочерних элементов
контейнера.

Syntax
------

.. code:: js

    container.getItems()

Returns
~~~~~~~

`Коллекция <../../../Collection/>`__ дочерних элементов контейнера.

Examples
--------

.. code:: js

    var items = container.getItems();
