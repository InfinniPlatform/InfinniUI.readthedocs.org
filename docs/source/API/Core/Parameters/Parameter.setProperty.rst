Parameter.setProperty
=====================

Устанавливает значение атрибута параметра.

Syntax
------

.. code:: js

    Parameter.setProperty(name, value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``name``
     - ``String``
     - Имя атрибута.
   * - ``value``
     - \*
     - Значение атрибута.


Returns
-------

Нет

Examples
--------

.. code:: js

    Parameter.setProperty('qty', 33);
