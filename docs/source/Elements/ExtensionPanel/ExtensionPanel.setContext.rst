Element.setContext()
====================

Устанавливает `контекст представления </Core/Context/>`__.

Syntax
------

.. code:: js

    Element.setContext(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - `ViewContext </Core/Context/>`__
     - Контекст представления


Returns
~~~~~~~

Метод не возвращает значений.

Examples
--------

.. code:: js

    Element.setContext(view.getContext());
