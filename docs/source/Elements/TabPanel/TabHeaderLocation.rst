TabHeaderLocation
=================

Расположение закладок в `TabPanel <./>`__.

Syntax
------

.. code:: js

    TabHeaderLocation = {
      none: 'None',
      left: 'Left',
      top: 'Top',
      right: 'Right',
      bottom: 'Bottom',
    }

Members
~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Description
   * - none
     - Закладки не отображаются
   * - left
     - Закладки отображаются слева
   * - top
     - Закладки отображаются сверху
   * - right
     - Закладки отображаются справа
   * - bottom
     - Закладки отображаются снизу
