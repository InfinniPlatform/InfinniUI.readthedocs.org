Button.getAction
================

Возвращает `действие <../../../Core/Actions/>`__ при нажатии на кнопку.

Syntax
------

.. code:: js

    button.getAction()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`Действие <../../../Core/Actions/>`__ при нажатии на кнопку.

Examples
--------

.. code:: js

    var action = button.getAction();
