DataNavigation.setPageSize
==========================

Устанавливает размер страницы.

Syntax
------

.. code:: js

    DataNavigation.setPageSize(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``Integer``
     - Размер страницы.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    dataNavigation.setPageSize(10);
