DataNavigation.setIsDataReady
=============================

Устанавливает значение которое показывает готовы ли данные.

Syntax
------

.. code:: js

    DataNavigation.setIsDataReady(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Готовность данных.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    dataNavigation.setIsDataReady(true);
