DataNavigation.getIsDataReady
=============================

Возвращает значение которое показывает готовы ли данные.

Syntax
------

.. code:: js

    DataNavigation.getIsDataReady()

Returns
~~~~~~~

true или false.

Examples
--------

.. code:: js

    var isDataReady = dataNavigation.getIsDataReady();

See Also
--------

-  `setIsDataReady() <DataNavigation.setIsDataReady.html>`__
