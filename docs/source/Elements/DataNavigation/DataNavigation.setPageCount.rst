DataNavigation.setPageCount
===========================

Устанавливает количество страниц.

Syntax
------

.. code:: js

    DataNavigation.setPageCount(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Number``
     - Количество страниц.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    dataNavigation.setPageCount(true);
