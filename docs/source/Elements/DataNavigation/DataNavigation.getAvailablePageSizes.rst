DataNavigation.getAvailablePageSizes
====================================

Возвращает `коллекцию <../../Core/Collection/>`__ доступных размеров
страниц.

Syntax
------

.. code:: js

    DataNavigation.getAvailablePageSizes()

Returns
~~~~~~~

`Коллекцию <../../Core/Collection/>`__ доступных размеров страниц.

Examples
--------

.. code:: js

    var pageSizes = dataNavigation.getAvailablePageSizes();
