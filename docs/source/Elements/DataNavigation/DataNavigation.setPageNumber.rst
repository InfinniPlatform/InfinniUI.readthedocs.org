DataNavigation.setPageNumber
============================

Устанавливает номер текущей страницы.

Syntax
------

.. code:: js

    DataNavigation.setPageNumber(value);

Parameters
~~~~~~~~~~

+-------------+---------------+---------------------------+
Name        | Type          | Description               |
+=============+===============+===========================+
``value``   | ``Integer``   | Номер текущей страницы.   |
+-------------+---------------+---------------------------+

Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    dataNavigation.setPageNumber(2);

--------
