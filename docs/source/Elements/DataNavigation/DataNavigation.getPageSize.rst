DataNavigation.getPageSize
==========================

Возвращает размер страницы.

Syntax
------

.. code:: js

    DataNavigation.getPageSize()

Returns
~~~~~~~

Размер страницы.

Examples
--------

.. code:: js

    var pageSize = dataNavigation.getPageSize();
