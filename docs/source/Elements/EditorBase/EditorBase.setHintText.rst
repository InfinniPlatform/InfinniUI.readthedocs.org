EditorBase.setHintText
======================

Устанавливает текст с подсказкой для ввода.

Syntax
------

.. code:: js

    EditorBase.setHintText(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``String``
     - Текст с подсказкой для ввода.


Examples
--------

.. code:: js

    EditorBase.setHintText('Enter positive value');
