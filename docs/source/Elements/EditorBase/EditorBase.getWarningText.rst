EditorBase.getWarningText
=========================

Возвращает текст с информацией о предупреждении.

Syntax
------

.. code:: js

    EditorBase.getWarningText()

Returns
~~~~~~~

Текст с информацией о предупреждении.

Examples
--------

.. code:: js

    var warningText = EditorBase.getWarningText();

--------
