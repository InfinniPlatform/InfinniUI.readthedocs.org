EditorBase.getErrorText
=======================

Возвращает текст с информацией об ошибке.

Syntax
------

.. code:: js

    EditorBase.getErrorText()

Returns
~~~~~~~

Текст с информацией об ошибке.

Examples
--------

.. code:: js

    var errorText = EditorBase.getErrorText();
