EditorBase.getHintText
======================

Возвращает текст с подсказкой для ввода.

Syntax
------

.. code:: js

    EditorBase.getHintText()

Returns
~~~~~~~

Текст с подсказкой для ввода.

Examples
--------

.. code:: js

    var hintText = EditorBase.getHintText();
