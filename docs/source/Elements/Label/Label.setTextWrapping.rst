Label.setTextWrapping
=====================

Устанавливает значение, определяющее, переносится ли текст при
переполнении.

Syntax
------

.. code:: js

    Label.setTextWrapping(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Значение, определяющее, переносится ли текст при переполнении.

Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    myLabel.setTextWrapping(false);

--------
