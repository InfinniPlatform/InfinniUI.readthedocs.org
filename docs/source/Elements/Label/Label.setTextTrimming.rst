Label.setTextTrimming
=====================

Устанавливает значение, определяющее, усекается ли текст при
переполнении.

Syntax
------

.. code:: js

    Label.setTextTrimming(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Значение, определяющее, усекается ли текст при переполнении.


Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    myLabel.setTextTrimming(false);

--------
