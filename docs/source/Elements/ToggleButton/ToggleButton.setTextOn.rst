ToggleButton.setTextOn
======================

Устанавливает текст на включенное состояние.

Syntax
------

.. code:: js

    toggleButton.setTextOn(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``string``
     - текст на включенное состояние.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    toggleButton.setTextOn("ON");
