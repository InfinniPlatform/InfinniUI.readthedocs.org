ToggleButton.getTextOff
=======================

Возвращает текст на выключенное состояние.

Syntax
------

.. code:: js

    toggleButton.getTextOff()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Текст на выключенное состояние.

Examples
--------

.. code:: js

    var text = toggleButton.getTextOff();

--------
