ToggleButton.getTextOn
======================

Возвращает текст на включенное состояние.

Syntax
------

.. code:: js

    toggleButton.getTextOn()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Текст на включенное состояние.

Examples
--------

.. code:: js

    var text = toggleButton.getTextOn();
