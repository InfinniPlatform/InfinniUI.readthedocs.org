ToggleButton.setTextOff
=======================

Устанавливает текст на выключенное состояние.

Syntax
------

.. code:: js

    toggleButton.setTextOff(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``string``
     - текст на выключенное состояние.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    toggleButton.setTextOff("OFF");
