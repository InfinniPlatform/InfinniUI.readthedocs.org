ScrollPanel.setVerticalScroll()
===============================

Устанавливает `видимость полосы прокрутки <ScrollVisibility.html>`__ по
вертикали.

Syntax
------

.. code:: js

    scrollPanel.setVerticalScroll(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - `ScrollVisibility <ScrollVisibility.html>`__
     - Видимость полосы прокрутки по вертикали


Returns
~~~~~~~

Метод ничего не возвращает.

Examples
--------

.. code:: js

    scrollPanel.setVerticalScroll(ScrollVisibility.auto);
