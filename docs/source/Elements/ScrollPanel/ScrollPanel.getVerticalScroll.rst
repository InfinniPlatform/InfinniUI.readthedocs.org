ScrollPanel.getVerticalScroll()
===============================

Возвращает `видимость полосы прокрутки <ScrollVisibility/>`__ по
вертикали.

Syntax
------

.. code:: js

    scrollPanel.getVerticalScroll()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`Видимость полосы прокрутки <ScrollVisibility/>`__ по вертикали.

Examples
--------

.. code:: js

    var verticalScroll = scrollPanel.getVerticalScroll();
