DataGridColumn.setSortable
==========================

Устанавливает или убирает возможность сортировки данной колонки.

Syntax
------

.. code:: js

    DataGridColumn.setSortable(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Возможность сортировки колонки.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    column.setSortable(true);
