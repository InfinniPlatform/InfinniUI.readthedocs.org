DataGridColumn.setIsHeaderTemplateEmpty
=======================================

Устанавливает boolean-значение о том, что есть шаблон ячейки заголовка.

Syntax
------

.. code:: js

    DataGridColumn.setIsHeaderTemplateEmpty(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``Boolean``
     - Устанавливает boolean-значение о том, что есть шаблон ячейки заголовка.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    DataGridColumn.setIsHeaderTemplateEmpty(true);

--------
