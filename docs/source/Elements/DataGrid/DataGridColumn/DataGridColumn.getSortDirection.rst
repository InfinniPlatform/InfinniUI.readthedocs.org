DataGridColumn.getSortDirection
===============================

Возвращает направление сортировки.

Syntax
------

.. code:: js

    DataGridColumn.getSortDirection();

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

'asc' или 'desc'.

Examples
--------

.. code:: js

    column.getSortDirection();
