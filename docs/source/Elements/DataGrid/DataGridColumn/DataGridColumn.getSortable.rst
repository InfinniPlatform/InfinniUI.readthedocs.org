DataGridColumn.getSortable
==========================

Возвращает boolean-значение, которое показывает можно ли применять
сортировку к данной колонке.

Syntax
------

.. code:: js

    DataGridColumn.getSortable();

Parameters
~~~~~~~~~~

Нет.

Returns
~~~~~~~

true или false.

Examples
--------

.. code:: js

    column.getSortable();
