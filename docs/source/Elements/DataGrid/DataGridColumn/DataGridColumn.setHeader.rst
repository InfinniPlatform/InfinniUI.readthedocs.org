DataGridColumn.setHeader
========================

Устанавливает заголовок колонки таблицы.

Syntax
------

.. code:: js

    DataGridColumn.setHeader(value);

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``String``
     - Заголовок колонки таблицы.


Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    column.setHeader("Id");
