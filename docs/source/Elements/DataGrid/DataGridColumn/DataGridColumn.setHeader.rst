DataGridColumn.setHeader
========================

Устанавливает заголовок колонки таблицы.

Syntax
------

.. code:: js

    DataGridColumn.setHeader(value);

Parameters
~~~~~~~~~~

+---------+--------------+------------------------------+
Name    | Type         | Description                  |
+=========+==============+==============================+
value   | ``String``   | Заголовок колонки таблицы.   |
+---------+--------------+------------------------------+

Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    column.setHeader("Id");

--------
