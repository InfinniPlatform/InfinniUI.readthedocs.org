DataGridColumn.isSortable
=========================

Возвращает true если колонку можно сортировать.

Syntax
------

.. code:: js

    DataGridColumn.isSortable();

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

true или false.

Examples
--------

.. code:: js

    column.isSortable();
