DateTimePicker.getMaxValue
==========================

Возвращает максимальное значение для ввода.

Syntax
------

.. code:: js

    DateTimePicker.getMaxValue()

Returns
~~~~~~~

Максимальное значение для ввода.

Examples
--------

.. code:: js

    var maxValue = dateTimePicker.getMaxValue();
