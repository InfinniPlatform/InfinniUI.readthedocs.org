DateTimePicker.getMinValue
==========================

Возвращает минимальное значение для ввода.

Syntax
------

.. code:: js

    DateTimePicker.getMinValue()

Returns
~~~~~~~

Минимальное значение для ввода.

Examples
--------

.. code:: js

    var minValue = dateTimePicker.getMinValue();

--------
