DateTimePicker
==============

Редактор даты и времени. Значением данного элемента всегда является
текстовая строка, представляющая собой выбранное значение даты и времени
в формате ISO 8601.

Extends
-------

`TextEditorBase <../TextEditorBase/>`__

Syntax
------

.. code:: js

    new DateTimePicker(parent)
