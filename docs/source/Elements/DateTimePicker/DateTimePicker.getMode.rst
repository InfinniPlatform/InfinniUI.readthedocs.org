DateTimePicker.getMode
======================

Возвращает режим работы элемента.

Syntax
------

.. code:: js

    DateTimePicker.getMode()

Returns
~~~~~~~

Режим работы элемента.

Examples
--------

.. code:: js

    var mode = dateTimePicker.getMode();
