NumericBox.getMaxValue
======================

Возвращает максимальное значение для ввода.

Syntax
------

.. code:: js

    NumericBox.getMaxValue()

Returns
~~~~~~~

Максимальное значение для ввода.

Examples
--------

.. code:: js

    var maxValue = numericBox.getMaxValue();
