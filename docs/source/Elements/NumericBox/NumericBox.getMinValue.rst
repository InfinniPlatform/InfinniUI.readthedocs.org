NumericBox.getMinValue
======================

Возвращает минимальное значение для ввода.

Syntax
------

.. code:: js

    NumericBox.getMinValue()

Returns
~~~~~~~

Минимальное значение для ввода.

Examples
--------

.. code:: js

    var minValue = numericBox.getMinValue();
