NumericBox.getStartValue
========================

Возвращает начальное значение по умолчанию.

Syntax
------

.. code:: js

    NumericBox.getStartValue()

Returns
~~~~~~~

Начальное значение по умолчанию.

Examples
--------

.. code:: js

    var startValue = numericBox.getStartValue();

--------
