NumericBox.setIncrement
=======================

Устанавливает приращение значения при вводе с помощью кнопок.

Syntax
------

.. code:: js

    NumericBox.setIncrement(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - ``value``
     - ``Number``
     - Приращение значения при вводе с помощью кнопок.


Examples
--------

.. code:: js

    numericBox.setIncrement(5);

--------
