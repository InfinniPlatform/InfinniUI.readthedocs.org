Link.setHref
============

Устанавливает адрес документа, на который должна переводить ссылка.

Syntax
------

.. code:: js

    Link.setHref(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - ``String``
     - Адрес документа, на который должна переводить ссылка.


Returns
~~~~~~~

Нет.

Examples
--------

.. code:: js

    link.setHref("http://docs.infinnity.ru/");

--------
