Link.getHref
============

Возвращает адрес документа, на который переводит ссылка.

Syntax
------

.. code:: js

    Link.getHref()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

Адрес документа, на который переводит ссылка.

Examples
--------

.. code:: js

    var href = link.getHref();

--------
