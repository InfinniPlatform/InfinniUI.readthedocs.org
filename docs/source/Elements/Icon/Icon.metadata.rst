Icon.metadata
-------------

Метаданные типа `Icon <./>`__.

Properties
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - Size
     - `TextStyle </Core/Style/TextStyle/>`_\*
     - Размер иконки

\* в данном случае `TextStyle </Core/Style/TextStyle/>`_ определяет только размер шрифта.

Examples
~~~~~~~~

.. code:: json

    {
        "Icon": {
            "Value": "star",
            "Size": "display1"
        }
    }
