Icon.getSize
================

Возвращает значение размера иконки.

Syntax
------

.. code:: js

    icon.getSize()

Parameters
~~~~~~~~~~

Нет

Returns
~~~~~~~

`TextStyle </Core/Style/TextStyle/>`__.

Examples
--------

.. code:: js

    var size = icon.getSize();

See Also
--------

-  `setSize <Icon.setSize.html>`__
