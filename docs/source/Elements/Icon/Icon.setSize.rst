Icon.setSize
================

Устанавливает размер иконки. 

Syntax
------

.. code:: js

    icon.setSize(value)

Parameters
~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - value
     - `TextStyle </Core/Style/TextStyle/>`__
     - Размер иконки


Returns
~~~~~~~

Нет

Examples
--------

.. code:: js

    icon.setSize(TextStyle.display1);
